# 所见网站，皆可为我所用－templatespider
<br/>

1. 看着网上哪个网站好，可用此软件将其下载，变为模版，可以供网市场云建站、帝国CMS、织梦CMS等各大建站系统使用<br/>
2. 下载的模版，采用人工智能，自动进行计算识别，将其转化，导出网市场云建站可使用的模版，做到一键导入使用！<br/>
